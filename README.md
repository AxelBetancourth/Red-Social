//English
In this project, I created a social network for a fictional company, where there is an initial interface to display the social network and a section to log in and register in case the user hasn't logged in yet. After logging in, it allows the user to access a section where they can make posts and view their profile, which displays their post history. I made this project when I was starting to program with backend, perhaps it didn't turn out the best possible, but it was my first web project where I put a lot of effort and it helped me grow as a programmer.//There is only the code, I have not deployed it yet//
